Recuerden que es una página web ESTÁTICA.
Cuenta con 2 propiedades básicas: Atributos y contenido.

"< p class= "texto"> CURSO_1 < /p>

`
  • "< >":Etiqueta
  • "class":nombre
  • "Valor":"texto"
  • "Contenido":CURSO_1
  • "< />":Etiqueta de cierre

ESTRUCTURA BÁSICA DEL HTML

"< html >
. . < head>
. . < title>
. . < /title>
. . < /head>
. . < body>
. . < /body>
< /html>


Se pueden agregar otros elementos que ayudan a personalizar nuestra página HTML, a continuacion se mencionarán algunos de ellos:

  • "< Hn >: Permite agregar un título o subtítulo
  • "< b >": Letra en negrita, tiene cierre
  • "< i >": Letra en cursiva, tiene cierre
  • "< u >": Letra con rayita abajo, tiene cierre
  • "< a >": Agregar una dirección, le acompaña href
  • "< a href="pagina" >": Te dirige a alguna página web, tiene cierre
  • "< target="_black" >": Dentro del anterior, te dirige a otra página generando una nueva pestaña
  • "< meta charset="TIPO" >": Letra en cursiva, tiene cierre
  • "< p >": Genera párrafos
  • "< ol >": Genera una lista ordenada, con números
  • "< ol >": Genera una lista ordenada, con números
  • "< ul >": Genera una lista desordenada, con íconos.
  • "< il >": Genera un nuevo item de la lista con "ul" o "ol"
  • "< table >": Abre una tabla de datos
  • "< tr >": Crea una fila de la tabla
  • "< td >": Crea una columna de la tabla
  • "< font >": Permite modificar las características de un texto
  • "< face="N" >": Define el tipo de letra dentro de font
  • "< color="N" >": Define el color dentro de font
  • "< size="#" >": Define el tamaño de letra dentro de font
  • "< img src="NOMBRE.tipo_de_imagen">": Agrega una imagen
  • "< height >": Ancho
  • "< width >": Largo
  • "< br >": Genera un espaciado, lo lleva a otra línea
  • "< hr >": Genera una línea horizontal, se puede modificar tamaño y color
  • "< form >": Method o action... genera algo
  • "< input >": Type="circle, button, radio, checkbox, text". Inserta según su tipo
  • "< select >": Permite seleccionar opciones
  • "< option >": Son las opciones XD
  • "< aling >": Dirige a que parte de la página se orienta el texto
